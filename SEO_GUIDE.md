## 1️⃣ Meta Tags المحسّنة

```html

<meta name="description" content="معرض أثاث متكامل يضم براويز حديد، ستيل، خشب، أبواب حماية، ديكورات وأثاث فاخر. جودة عالية وأسعار منافسة.">

```

### صفحة المنتجات (products.html)

## 2️⃣ Schema Markup (البيانات المنظمة)

### Organization Schema
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "النور للحديد",
  "description": "معرض أثاث متكامل",
  "url": "https://ahmed19992489.github.io/ELNOUR-FOR-STEEL/",
  "telephone": "+201114323218",
  "address": {
    "@type": "PostalAddress",
    "addressCountry": "EG"
  }
}
```

### Product Schema
```json
{
  "@context": "https://schema.org/",
  "@type": "Product",
  "name": "براويز ماندالا فاخرة",
  "description": "براويز حديد مقصوصة بالليزر بدقة عالية",
  "offers": {
    "@type": "Offer",
    "price": "3950",
    "priceCurrency": "EGP"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "5",
    "reviewCount": "24"
  }
}
```

## 3️⃣ Open Graph Tags

تم إضافة Open Graph Tags لتحسين مشاركة الموقع على وسائل التواصل:

```html
<meta property="og:type" content="website">
<meta property="og:title" content="النور للحديد - معرض أثاث متكامل">
<meta property="og:description" content="براويز حديد، ستيل، خشب، أبواب حماية وديكورات فاخرة">
<meta property="og:url" content="https://ahmed19992489.github.io/ELNOUR-FOR-STEEL/">
<meta property="og:image" content="[صورة الموقع]">
<meta property="og:site_name" content="النور للحديد">
```

## 4️⃣ Sitemap و Robots.txt

### Sitemap.xml
- ✅ يحتوي على جميع صفحات الموقع
- ✅ يتضمن أولويات الصفحات (Priority)
- ✅ يحدد تكرار التحديث (Changefreq)
- ✅ يتضمن Mobile URLs

### Robots.txt
- ✅ يسمح لمحركات البحث بالزحف الكامل
- ✅ يحدد موقع Sitemap
- ✅ يتضمن قواعس محددة لمحركات البحث المختلفة
